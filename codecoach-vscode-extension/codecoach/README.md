# CodeCoach

CodeCoach is a VS Code extension that helps students learn by giving hints and explanations (not full solutions) inside a sidebar chat.

## Features
- Sidebar chat view
- Paste a Gemini API key to enable coaching responses
- Deep-link support from the CodeCoach web app (vscode://... links)
- Export a learning summary

## Setup
1. Install the extension (.vsix)
2. Run **CodeCoach: Set Gemini API Key**
3. Open the CodeCoach sidebar and start chatting

## Demo mode
You can enable demo mode in settings: `codecoach.demoMode`.

## Privacy
Messages are sent to the configured model provider only when you use the chat.
